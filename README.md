# MusicPlayer
一个基于Elctron-vite+Vue3构建的本地音乐播放器

效果演示
![image1](./public/img1.png)

![image2](./public/img2.png)

![image3](./public/img3.png)

在MusicPlayer中，你可以导入自己的本地音乐。
并按照你的喜好来移动歌曲在列表中的顺序。
在播放时可以选择随机播放与顺序播放。
> 温馨提示：歌词需要在联网状态下才能加载

# 快速开始

安装依赖
```bash
npm install
```

运行
```bash
npm run dev
```

## 打包
```bash
npm run build:win
```
